# PatientMangementSystem

     Tech Stack Used
       1. Node Js (Backend)
       2. React Js (Frontend)
       3. Google Bucket For Image Storage.
       
# For Setup 
git clone  https://github.com/ashishpokhrel123/PatientMangementSystem

(Backend)
cd api/src
npm install
create google storage bucket
npm start



 (frontend)
cd client/pms
npm install
npm start
